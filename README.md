MelCNN
------

Blind sound source separation of multiple speakers on a single channel with Dilated Convolution.


## Installation
```sh
# clone
$ git clone <this repo>
$ cd <this repo>

# create venv
$ python -m venv venv
$ source source venv/bin/activate

# install python libs
$ pip install --upgrade pip
$ pip install -r requirements.txt
```
